# Task Decomposition Engine


## Tech Stack

- **Backend:** Node.js, Express
- **Frontend:** React
- **Other:** Fetch API for frontend-backend communication

---

## Project Setup

### 1. Clone the Repository

```bash
git clone https://github.com/ShahbazAhmad08/VAB_Informatics_Assignmen.git


```
### 2. Run Backend
```bash
cd VAB_Informatics_Assignmen
cd backend
npm install
node server.js
```
### 2. Run Frontend
```bash
cd VAB_Informatics_Assignmen
cd frontend
npm install
npm start
```
